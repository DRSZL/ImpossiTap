# ImpossiTap - Micro Edition 🎯

Ein minimalistisches High-Precision Game. Triff die Mikrosekunde!

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- Messung: Performance API (Microseconds)
- Geplant: Capacitor für iOS/Android & Firebase Leaderboard

## Installation
1. Repository klonen.
2. `index.html` im Browser öffnen.
